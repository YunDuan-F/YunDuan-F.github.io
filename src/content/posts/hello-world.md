Welcome to [Astro](https://astro.build/) and [Fuwari](https://github.com/saicaca/fuwari). This post was migrated from the previous Hexo blog.

## Quick Start

### Create a new post

```bash
pnpm new-post
```

### Run server

```bash
pnpm dev
```

### Generate static files

```bash
pnpm build
```

### Preview production build

```bash
pnpm preview
```
